# DockerMEAN

# Description:
Allows the user to run the frontend, backend and mongo through Docker

# To Run:
  - Build Images  :   docker build -d <dockerName/imageName:versionName> .
  
  - MultiVM       :   terraform apply
  
  - OneVM         :   docker-compose up -d
